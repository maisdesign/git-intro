# git-intro

Questo repository contiene un semplice esempio introduttivo all'uso di Git e GitHub.

## Contenuto
- `index.html`: Un file HTML di esempio per iniziare a lavorare con Git.

## Come iniziare
1. Clona il repository:
   ```bash
   git clone https://github.com/maisdesign/git-intro.git
   ```
2. Apri il file `index.html` con il tuo editor di testo preferito o direttamente nel browser.

## Obiettivi
- Imparare i comandi base di Git (clone, add, commit, push, pull)
- Comprendere il flusso di lavoro con GitHub
- Eseguire modifiche e gestire versioni di file

## Risorse utili
- [Guida ufficiale Git](https://git-scm.com/doc)
- [GitHub Docs](https://docs.github.com/it)
